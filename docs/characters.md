這份角色設定彙整了「反智特攻隊 (The Glitch Squad)」的六位核心成員、玩家角色以及移動基地。這些角色並非單純的丑角，而是設計為**「照妖鏡」**，反映出每個人在極端情境下都可能滑向的陰暗面，讓玩家在厭惡他們的同時產生「這跟我某個時刻好像」的共鳴。
以下是針對 GitHub 輸出的系統化角色檔案：
0. 玩家：維修工 (The Mechanic)
• 定位： 戰術指揮官 / 唯一清醒的人。
• 核心特質： 相信「祛魅 (Disenchantment)」，認為世界只有物理法則與因果律。
• 戰鬥機制： 不直接攻擊，而是透過 QTE 介入來修正隊友的錯誤動作、增加命中或減少傷害。
• 角色意義： 扮演疲憊的家長，試圖在崩壞的廢墟中修好機器，也修好這群隊友的腦袋。
1. 肯特 (Kent)
• 原型： 地平論者 / 陰謀論大師。
• 屬性： 摩羯座 (Capricorn) / ISTP-T / B 型。
• Glitch (性格缺陷)： 視野鎖定 (Tunnel Vision)。他不相信沒親眼看到的東西，認為痛苦與子彈都是政府的全像投影。
• 戰鬥機制： 正面防禦極高（因為他盯著威脅看並否定其存在），但側面與背面傷害加倍。
• 照妖鏡： 反映出我們體內那種「只相信自己邏輯、拒絕承認事實」的固執。
2. 查德 (Chad)
• 原型： 傲慢現充 / 天龍人 / 紅二代。
• 屬性： 獅子座 (Leo) / ESTJ-A / AB 型。
• Glitch (性格缺陷)： 繼承型實力。將父親提供的特權誤以為是自己的能力，深信世界繞著自己轉。
• 戰鬥機制： 資源揮霍。攻擊不消耗 MP 而是消耗玩家的庫存物資（錢與彈藥）；順風時暴擊率極高，受傷時會惱羞成怒或賣隊友。
• 照妖鏡： 反映出「追求效率、把別人當 NPC、將好運誤認作實力」的優越感。
3. 巴斯特 (Buster)
• 原型： 交易型人格 / 崩潰的好學生 / 小草。
• 屬性： 處女座 (Virgo) / ISFJ-T / A 型。
• Glitch (性格缺陷)： 期望值管理失敗。深信「我聽話 = 該有回報」，一旦受挫就會覺得世界不公並陷入怨恨。
• 戰鬥機制： 乖寶寶印章。成功執行指令可疊加攻擊力；若受傷或沒被補血，印章歸零並進入 [怨恨] 狀態，拒絕行動或隨機攻擊隊友。
• 照妖鏡： 反映出「不想努力、只想被照顧、一不如意就覺得被全世界針對」的受害者心理。
4. 艾蜜莉 (Emili)
• 原型： DEI 審判官 / 情緒勒索者。
• 屬性： 天秤座 (Libra) / ENFJ-A / A 型。
• Glitch (性格缺陷)： 強制均等。無法容忍任何差異或特權，認為感受高於事實。
• 戰鬥機制： 再分配 (Redistribution)。她無法單體補血，只能將全隊血量平均化，這在應對範圍攻擊時非常危險。
• 照妖鏡： 反映出「用關係換取特權、用道德綁架團隊、用弱勢作為武器」的索求行為。
5. 達莉亞 (Daria)
• 原型： 虛無主義者 / 躺平族 / 高智商廢人。
• 屬性： 水瓶座 (Aquarius) / INTP-T / AB 型。
• Glitch (性格缺陷)： 習得性絕望。深信「不做 = 不輸」，認為所有努力都是無用功。
• 戰鬥機制： 消極力場。被動降低全場行動速度；每回合有 25% 機率因為「正在滑手機」而拒絕行動。
• 照妖鏡： 反映出我們體內那個「因為害怕失敗而拒絕開始、嘲笑他人熱情」的冷漠自我。
6. 露絲 (Ruth)
1. MBTI：ISTJ-T (物流師 / 動盪型)
	•	根據： 文本裡直接點名了 ISTJ，而且是 T (Turbulent/動盪) 分數拉滿的那種。
	•	分析：
	◦	Si (內向實感) 主導： 她極度依賴「過去的經驗」和「手冊（SOP）」。對她來說，C++ 手冊就是絕對真理，因為那是已知最安全的規矩。
	◦	Ne (外向直覺) 劣勢 (The Grip)： 這就是文中提到的「災難化思考」。一般 ISTJ 只是保守，但壞掉的 ISTJ 會把「未知的變數」無限放大成「世界末日」。
	◦	行為對應： 看到 Buster 拿重物，她不是看到「他在重訓」，她是看到「脊椎側彎 -> 神經壓迫 -> 半身不遂 -> 變拖油瓶」的連鎖反應。
2. 血型：A 型 (而且是那種超神經質的 A 型)
	•	根據： 典型的 A 型特質是認真、細膩、守規矩、責任感強，但缺點就是 「容易操心過頭」 和 「悲觀主義」。
	•	分析：
	◦	O 型會說「沒事啦衝過去」，B 型會說「我們來試試看這樣跳」，只有 A 型會說「橋體共振頻率測過了嗎？」。
	◦	A 型的強迫症體現在她把隊友包成「肉粽」。她不是為了贏，她是為了「零失誤」。這完全符合 A 型人那種「如果我不把細節顧好，天就會塌下來」的心理壓力。
3. 星座：處女座 (Virgo)
	•	根據： 只有處女座能把「碎碎唸」昇華成一種愛的表現，並且具備醫療/健康相關的焦慮（Hypochondria）。
	•	分析：
	◦	細節控： 她檢查盔甲縫隙、翻閱手冊指著 return 0; 說安全的樣子，完全是處女座在追求完美的秩序感。
	◦	健康焦慮： 處女座掌管健康宮位。她罵 Chad 盔甲反光會引導子彈穿過「頸動脈」、罵 Buster 會「半身不遂」，這種對生理結構受傷的具體想像，非常處女座。
	◦	服務型人格： 雖然嘴巴毒（碎唸），但手一直在補血/上 Buff。處女座就是這種「一邊嫌你髒、一邊幫你洗內褲」的勞碌命角色。

2. 戰鬥機制：【血肉獻祭】
• 代價機制： 她所有的強大治療與 Buff 技能都不消耗 MP，而是扣除她自己的 HP。她越殘血，效果就越強。

7. 移動基地：慈悲號 (The Mercy)

💡 養成邏輯說明： 每個角色擁有個別的 「依賴 (Dependency) vs. 覺醒 (Awakening)」 滑桿。
• 依賴路線： 角色變聽話，QTE 判定變輕鬆，最終成為穩定的「指令執行機器」。
• 覺醒路線： 雖然不受控，但會產生「神來一筆」的神操作或隱藏技能，最終成為「不可控的天才」。
💡 類比 (Analogy)： 這群隊友就像是一組參數錯誤的 3D 列印機。你身為維修工（玩家）的工作不是更換機器，而是透過不斷的校準（戰鬥 QTE）與設定調整（營地對話），讓這些「瑕疵品」在即將瓦解的烏托邦中印製出生存的希望。
