# 魔法系統名稱：【理型具現】 Logos Manifestation
1. 核心原理：知識即力量
能量來源（熵值暴走）： 世界充滿了混亂的「想像能量（熵）」，雖然無窮無盡，但若無引導則會導致現實崩塌或使用者自爆。

邏輯導管（學術根基）： 魔法的本質是透過**「學術模型」**來約束能量。施法者必須理解該現象的物理、化學或醫學原理，才能將想像實體化。

例如：想放火球，腦中必須瞬間構建熱力學與化學反應模型。

邏輯檢定（真理性）： 魔法不成立的主要原因是「邏輯不通」。若想像內容違反使用者已知的物理規律或缺乏學理支撐，魔法將無法發動或產生反噬。

2. 能量指標 MP 與 HP 的代價
MP Magic Point —— 「神經緩衝電位」：

代表大腦在不產生物理損傷的情況下，處理「邏輯運算」的安全容量。

消耗邏輯： 對學理愈精通（等級愈高），邏輯摩擦力愈小，消耗 MP 愈少。

回復： 透過神經冷卻、休息或使用「神經冷卻液」（Ether）回復。

HP (Hit Point) —— 「生命超頻代價」：

當 MP 耗盡，施法者可選擇強行施法。此時能量會直接燒灼神經元與身體組織（超頻）。

風險： 雖然威力強大，但會導致實質的身體受損（熱灼傷、組織壞死），且血量過低時極易被敵人一擊必殺。

3. 實作與成長（玩家的努力感）
魔石系統 (Magicite)： 儲存了高階學術模型或古代演算法的資料載體。透過「同步」魔石，施法者可以快速學會複雜的魔法（如高級醫學或量子物理）。

艾德加式機械 (Tools)： 為了節省 MP/HP，聰明的施法者會搭配 3D 列印製造的實體機械。

策略：利用機械處理物理部分，魔法僅用於關鍵的動力或參數改寫。

升級路徑：

唸書學習： 攻讀醫學、物理、工程等學科，優化施法效率。

硬體改裝： 透過 3D 列印優化散熱器、擴充神經緩衝器（增加 MP）。

4. 戰鬥美學：邏輯博弈
沒有失敗率： 只要有能量，魔法一定會發動，但威力與消耗取決於使用者的學識。

邏輯瓦解： 戰鬥中可透過「指出對方的邏輯漏洞」來削弱或抵消敵方的魔法（例如：識破對方的重心不穩或材質缺陷）。

醫學應用： 從初階的「細胞增生縫合」進化到高階的「神經訊號阻斷」。

🌍 世界觀背景（Utopiapunk）
上城（壟斷者）： 擁有完美的知識與邏輯模型，但缺乏想像力與「熵」的驅動。

下城（江湖）： 能量充沛但缺乏教育。主角作為擁有上城邏輯與下城覺悟的「工匠」，在其中尋找真正的真理。
