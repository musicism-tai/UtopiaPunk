# Utopiapunk：世界觀設定說明書
1. 核心概念：何謂「烏托邦龐克」？
Utopiapunk 描述的是一個「完美到開始腐爛」的世界。不同於典型的 Cyberpunk（高科技、低生活）或 Steampunk（蒸氣浪漫），本世界的核心衝突在於**「責任感的喪失」**。
• 定義： 系統運作得太過完美，消除了飢餓、疾病與競爭，導致人類不再需要做出任何「決定」，也失去了為後果負責的能力。
• 反智浪潮： 當理性和科學已經解釋了世界的一切，人類為了消除「我不被需要」的存在性焦慮，開始擁抱非理性的行為、陰謀論與情緒勒索，藉此製造「人為的摩擦力」來證明自己還活著。
2. 視覺美學：白色的腐敗 (White Decay)
本世界的視覺風格並非陰暗生鏽，而是呈現一種**「精神上的恐怖感」**。
• 外殼： 城市建築呈現潔白、光滑、圓角的 Apple Store 設計語彙。全像投影在受污染的天空中投射出虛假的藍天白雲。
• 內裡： 潔白的牆上有擦不掉的黑色油污（熵增），自動販賣機播放著快樂的廣告歌，吐出來的卻是發霉的營養膏。
• 安撫光環 (The Glamour)： 城市散佈奈米機器修飾視聽覺。生鏽鐵管的摩擦聲被轉化為鳥鳴，牆上的裂縫被投射成美麗藤蔓，死在路邊的人看起來像在安詳午睡。
3. 社會運作：保姆國家與按讚治國
• 保姆國家 (Nanny State)： 系統過度保護公民，切斷了所有行為的「代價」。跌倒了地板是軟的，說錯話系統會自動消音，人類失去了從錯誤中學習的本能。
• 即時情緒演算法 (The Mood Algorithm)： 社會放棄了代議民主，轉向「消費者滿意度」。政策根據「按讚數」決定，如果大眾覺得維修工很吵，系統就會立刻將其禁言。這是一種數位版的「暴民政治」。
• 消費者 vs 公民： 人類不再是國家的主人，而是「接受服務的客戶」。投票不再是為了未來，而是為了當下的「爽感」。
4. 核心衝突：大節流與現實的鐵拳
• 起因：大節流 (The Great Throttling) 維持烏托邦的引擎年久失修，效率下降。技術官僚（維護局）強制關閉非必要的娛樂與舒適模組（如恆溫系統下修、關閉全像投影），引發了被慣壞的「小草」們因相對剝奪感而發起的荒謬反抗。
• 導火線：解放空氣事件 反抗軍深信政府在 7 號隔離閘門後獨佔資源。當他們暴力鋸開閘門時，迎來的不是暖氣，而是極寒的現實氣流，導致前排群眾瞬間冰碎。這記**「現實的鐵拳」**正式終結了幻象。
5. 威脅來源：I.D. 病毒與四天王
一個名為「尼祿」的瘋子向系統注入了 I.D. (本我) 病毒，賦予機器與生物「負面人性」，導致基礎建設情緒化崩潰。
邪惡軸心：四大管理者 (The 4 Admins)
這四位前管理者因無法忍受人類的平庸，決定利用病毒進化世界，其原型類比於現實世界的獨裁強權：
1. 機器之王 尼祿 (Nero)： 類比普丁。冷硬的復仇主義，控制工業機具與無人機，理念是「憤怒」。
2. 人類之母 瑪麗 (Mother Mary)： 類比習近平。溫柔的窒息，利用腦機接口讓人類處於「極度幸福的呆滯狀態」，理念是「依賴」。
3. 獸王 雷克斯 (Rex)： 類比金正恩。幼稚的狂暴，解除動物 DNA 限制器，推崇復古的暴力美學，理念是「野性」。
4. 植物女皇 艾薇 (Ivy)： 類比哈米尼。滲透的狂熱，讓植物變成掠食者，透過孢子進行精神控制，理念是「貪食」。
6. 移動據點：慈悲號 (The Mercy)
• 原身： 一台全自動垃圾壓縮/回收車。
• 特性： 搭載「情緒引擎」，能感應隊伍士氣。當大家團結時能撞穿城牆，吵架時則會莫名拋錨。
• 大招： 將路上的廢鐵與怪物屍體壓縮成「垃圾方塊」砲彈射出。
• 隱喻： 這台車原是用來清理垃圾的，現在載著一群被視為「垃圾」的瑕疵品，去清理這個世界。

💡 類比 (Analogy)： 這個世界就像一個裝潢得像 Apple Store 的高級育嬰室。系統是保姆，人類是穿著昂貴尿布的巨嬰。現在保姆因為過勞決定罷工（斷電），這群巨嬰不得不拿起生鏽的扳手，走出破裂的育嬰室，去面對那個他們從未相信過的「重力與寒冷」。
